## Injamuri Arun Kumar: Diff bet render and HttpResponse

| render() | HttpResponse() |
|----------|----------|
| Renders HTML templates | Returns raw content |
| Uses template engine | No template rendering |
| Commonly used for webpages | Commonly used for plain text or custom responses |

```python
from django.shortcuts import render

def home(request):
    return render(request, "index.html")
```

```python
from django.http import HttpResponse

def home(request):
    return HttpResponse("Hello World")
```

## M Harivardhan Reddy: Does Node.js automatically render code changes in the file

No.

Node.js itself does not automatically reload when files change.

Tools like **Nodemon** watch files and restart the server automatically.

```bash
npm install -g nodemon
```

```bash
nodemon app.js
```

## Naman Sharma: What is QuerySet in Django

A QuerySet is a collection of database records returned by Django ORM.

```python
users = User.objects.all()
```

```python
active_users = User.objects.filter(is_active=True)
```

QuerySets are lazy, meaning queries are executed only when needed.

## Tumma Haritha: What is ORM

ORM stands for **Object Relational Mapping**.

It allows interacting with a database using Python objects instead of SQL.

Without ORM:

```sql
SELECT * FROM users;
```

With ORM:

```python
User.objects.all()
```

Benefits:

- Less SQL code
- Database independent
- Easier maintenance
- More readable code

## Vikas Mehta: Diff bet Django Model and Record

| Model | Record |
|---------|---------|
| Blueprint/schema of a table | Single row in the table |
| Defined as a Python class | Instance of the model |
| Describes structure | Stores actual data |

Example:

```python
class User(models.Model):
    name = models.CharField(max_length=100)
```

Model = `User`

Record:

```python
User.objects.create(name="Musharaf")
```

Represents one row in the database.
