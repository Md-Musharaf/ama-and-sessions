## 1. Adhikya Edammala: Difference between em and rem

* em → Relative to the parent element's font size.
* rem → Relative to the root (html) font size.


## 2. Allanki VV Manikanta Sai: Difference between padding and margin

* Padding → Space inside the element (between content and border).
* Margin → Space outside the element (distance from other elements).

## 6. Kamparapu Lakshman: How meta tag is used?

* meta tags provides information about the webpage.

* Common usage: - Character encoding, Responsive design, SEO description

```
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Learning HTML">
```
